# F1gem
A gem to show finding results repository github on your terminal.
## Installation
Add this line to your application's Gemfile:

```ruby
gem "f1gem"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install f1gem

## Usage
`f1gem + keyword`
